# 📍 PlacesFinder

A modern, production-ready location-based web app built with React + Firebase. Discover restaurants, hotels, cafes, hospitals, and more using interactive maps powered by OpenStreetMap.

![PlacesFinder](https://via.placeholder.com/1200x600/0F0F0E/00D4A8?text=PlacesFinder)

---

## ✨ Features

- 🔍 **Place Search** — Instant search via Nominatim API
- 🗺️ **Interactive Maps** — React Leaflet + OpenStreetMap tiles
- 📍 **User Geolocation** — Find nearby places via browser location
- 🏷️ **12 Categories** — Restaurants, Hotels, Cafes, Hospitals, Pharmacies, Banks, ATMs, Gas Stations, Supermarkets, Schools, Parks, Tourist Attractions
- ❤️ **Favorites** — Save & manage favorite places (Firestore)
- ⭐ **Reviews & Ratings** — Rate and review any place
- 🔐 **Authentication** — Email/password + Google Sign-In
- 🕐 **Search History** — Track and replay searches
- 🌙 **Dark / Light Mode** — Persisted in localStorage
- 📱 **Fully Responsive** — Mobile-first design
